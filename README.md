## Deployment

### Deploy to Render

1. Push your code to GitHub
2. Connect your repository to Render
3. Select "Static Site" as service type
4. Deploy!

Render will automatically detect the static files and serve them.

### Local Development

Simply open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000`
